# Design Intelligence Gate

Pass only when existing design evidence was extracted and traced to authoritative files before new recommendations were made. Fonts, colors, typography, icons, strokes, radii, spacing, motion, theme, responsive, accessibility, and bundle impact must be classified as extracted, inferred, or proposed.

Fail on unowned recommendations, new libraries without bundle analysis, page-local tokens that duplicate the source of truth, arbitrary icon families/strokes, missing RTL/theme coverage, or claims based only on a generated report.
