# Testing And Observability

Test query serialization, debounce/cancellation, pagination/cursor boundaries, filter operators, sorting nulls and locales, page versus dataset aggregates, column visibility, selection, virtualization, loading/empty/error states, authorization, tenant isolation, exports, reconnects, event ordering, deduplication, and stale refreshes.

Observe request duration, database duration, rows scanned/returned, payload bytes, cache hit rate, render duration, long tasks, dropped events, reconnect count, refresh failures, and user-visible stale age. Correlate frontend requests, backend queries, and realtime events without logging secrets or sensitive row data.
