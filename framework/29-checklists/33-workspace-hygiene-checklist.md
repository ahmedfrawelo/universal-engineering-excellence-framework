# Workspace Hygiene Checklist

- [ ] Logs, traces, HAR files, temporary files, screenshots, coverage, and test reports are classified.
- [ ] Retention days are explicit and configurable.
- [ ] Dry run is the default.
- [ ] Source, configuration, secrets, Git metadata, release output, and tracked files are protected.
- [ ] Cleanup scripts live in the repository and work on the supported operating systems.
- [ ] A post-cleanup dry run and `git status` check pass.
