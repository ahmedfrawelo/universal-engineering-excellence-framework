# Skill Invocation Protocol Checklist

- [ ] Named user-requested skill checked.
- [ ] Installed skills checked.
- [ ] Project-local skills checked.
- [ ] UEEF task packs selected.
- [ ] Skill chain ordered by discovery, implementation, verification, and review.
- [ ] Red flags reviewed and corrected.
- [ ] TDD or equivalent evidence loop chosen.
- [ ] Subagent ownership bounded when delegation is used.
- [ ] External sources attributed.
- [ ] Final response reports only verified evidence.
