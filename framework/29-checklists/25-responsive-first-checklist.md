# Responsive First Checklist

Version: 1.1.0

Use this checklist with repository evidence. Marking an item complete requires a command, test, trace, screenshot, review record, or direct implementation reference.

- [ ] Content priority defined before desktop composition
- [ ] 320px width, tablet, desktop, landscape, 200 percent zoom, and short height checked
- [ ] No accidental horizontal overflow or covered controls
- [ ] Forms, tables, dashboards, media, and overlays have explicit strategies
- [ ] Touch targets, text scaling, localization, and keyboard order verified
