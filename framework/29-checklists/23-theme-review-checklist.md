# Theme Review Checklist

Version: 1.1.0

Use this checklist with repository evidence. Marking an item complete requires a command, test, trace, screenshot, review record, or direct implementation reference.

- [ ] Theme source and token source identified
- [ ] Light, dark, and system resolution verified
- [ ] Semantic colors, radius, type, spacing, shadow, and focus tokens used
- [ ] No incorrect-theme flash on supported boot paths
- [ ] Charts, forms, tables, overlays, and status states verified
