# Smart Environment Bootstrap and Dependency Manager

Version: 1.4.0

Verifies and repairs the engineering environment before any engineering task. Core and AI are mandatory; Frontend, Backend, Database, UIUX, DevOps, and Optional profiles are selected from evidence. Mandatory gaps block, Recommended gaps warn, and Optional gaps continue. For UI/UX tasks, ui-ux-pro-max and impeccable are both mandatory together.

## Runtime

Run scripts/environment-bootstrap.ps1 or scripts/environment-bootstrap.sh before project inspection. Never claim READY without current output.
