# Typography And Fonts

Identify the actual font files, language coverage, fallback stack, text roles, sizes, weights, line heights, letter spacing, and readable measure. Prefer an existing local font and stack when it supports the product's scripts; add a font only after checking licensing, loading cost, Arabic/Latin coverage, and rendering quality.

Use semantic roles such as display, heading, body, label, metadata, data-table, code, and numeric emphasis. Keep a coherent type scale and verify text scaling, truncation, wrapping, and RTL behavior.
