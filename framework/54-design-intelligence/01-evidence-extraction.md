# Evidence Extraction

Extract and review font files and `font-family` declarations, color literals and semantic CSS variables, typography sizes and weights, radius and stroke declarations, icon imports and assets, theme selectors, spacing values, shadows, motion durations, and responsive breakpoints.

The extractor is a discovery aid, not a substitute for semantic review. Group raw values into roles and identify the authoritative token file, component registry, and existing consumers before recommending changes.
