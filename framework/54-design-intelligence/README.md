# 54-design-intelligence

This pack extracts a project's existing design language before recommending anything new. It covers fonts, typography roles, colors, semantic tokens, icons and icon libraries, stroke/outline weights, radii, spacing, shadows, motion, themes, and responsive evidence.

The default behavior is evidence-first: reuse what exists, extend the existing source of truth when a role is missing, and recommend a new value only with a stated rationale, accessibility check, ownership, and bundle/runtime impact.
