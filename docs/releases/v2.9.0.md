# UEEF 2.9.0

Release date: 2026-07-16

## Skeleton loading hardening

- Added a shared render-timing contract with reveal delay, minimum visible duration, timeout, cancellation, stale-request handling, and deterministic timing tests.
- Added SSR, SSG, pre-rendering, server-component, hydration, suspense, and streaming-boundary guidance that prevents unstable server/client structure and content-to-skeleton regressions.
- Added a shared skeleton API and component-registry contract for governed primitives, proven recipes, public exports, ownership, tests, adoption, and deprecation.
- Strengthened the skeleton gate, checklist, contract template, runtime preflight, and cross-platform validators to reject flicker, arbitrary local timers, hydration mismatches, and duplicate page-local primitives.
