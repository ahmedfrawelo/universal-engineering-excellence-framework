# UEEF 2.14.0

Release date: 2026-07-23

## Deeper optional workflow tools

Project memory now supports task labels, query filtering, and a human-approval-only summary. Diff impact adds textual owner, contract, and import/require signals while retaining explicit heuristic limitations. Evidence export supports a non-writing preview plus optional memory and preflight context.

## Unix and CI

Unix now has bounded preflight, project-memory, and evidence-export commands. Unix preflight explicitly reports capability health as unsupported rather than claiming a probe result. Ubuntu CI runs the selected Unix workflow tool tests.

## Static callable readiness

Capability health can report `CALLABLE` only for an explicitly registry-bound skill whose local `SKILL.md` exists and whose exact provider plugin is enabled. This is static local readiness—not MCP startup, network reachability, or session selection.
