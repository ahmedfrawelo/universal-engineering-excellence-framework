# UEEF 2.8.3

Released: 2026-07-12

## Summary

This patch prevents minimized or background user-owned Chrome windows from being treated as a browser-control blocker.

## Changes

- Extension control continues for minimized, background, and non-foreground existing user tabs.
- Codex preserves the user's window state and does not restore, focus, resize, maximize, or move it.
- A visual-only capability failure defers only that visual gate; implementation and other verification continue.
- Generated loader and global instructions no longer blanket-block a banner on a proven existing claimed tab.
- Browser contract tests reject stale banner-only and minimized-window blocking rules.
