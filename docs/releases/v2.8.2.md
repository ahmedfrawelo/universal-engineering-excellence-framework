# UEEF 2.8.2

Released: 2026-07-12

## Summary

This patch aligns UEEF browser control with the supported Chrome extension workflow for the user's existing visible window.

## Changes

- Chrome tasks enumerate user tabs with `user.openTabs()` and claim the exact matching object with `claimTab()`.
- Extension attachment to an existing user tab is distinguished from a connector-created automation browser.
- Visible Windows control is a fallback only when the Chrome plugin is unavailable.
- Automation indicators are classified by provenance instead of causing an unconditional block.
- Default environment bootstrap now uses a mutable profile collection.
- Browser-policy and bootstrap behavior have executable regression tests.
