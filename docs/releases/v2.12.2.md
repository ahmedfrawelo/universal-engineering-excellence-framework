# UEEF 2.12.2

Release date: 2026-07-23

## Highlights

- Makes the existing Superpowers-inspired Pack 59 operational through `workflowDecisions` in the capability profile output.
- Each selected workflow records an ID, proportional selection level, task trigger, and expected evidence. The prior `workflows` list remains for compatibility.
- Simple self-contained work receives no workflow ritual; the selector escalates only for ambiguity, reproducible failures, behavior changes, or high-impact work.

## Validation

- Capability-profile tests cover ambiguity, regression debugging, production migration, UI, browser, and simple requests.
- Script syntax and framework validation remain required before runtime synchronization.
