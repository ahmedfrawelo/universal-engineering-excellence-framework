# UEEF 2.16.0

Release date: 2026-07-23

- Made the user's requested scope authoritative over continuation, delegation, autonomy, audits, and modernization. Bounded work now stops and answers when complete.
- Restored a single-agent default for narrow `T1` changes; delegation requires an independent benefit, while `T4` retains independent verification.
- Replaced the universal hard `medium` reasoning ceiling with proportional, recorded escalation for high-ambiguity `T3/T4` work.
- Prevented autonomous dependency inventory and upgrades for `T0/T1`, and made bootstrap, Guardian, and browser procedures proportional to explicit task triggers.
- Browser control now requires explicit browser/site/visual intent or a direct verification need; casual mentions do not select browser MCPs.
- Added intent-fidelity regression tests and fixtures, Ask/Do guidance, a visible route line, `strict-scope` team policy, and an installed-runtime source-revision warning.
