# UEEF 2.13.1

Release date: 2026-07-23

## Documentation and runtime integration

This patch makes the optional 2.13 workflow tools discoverable without turning them into a universal checklist. `QUICK_START.md`, the source loader, generated Codex loader, and generated `AGENTS.md` now direct agents to use task preflight for uncertainty, heuristic diff impact for multi-file changes, project memory for explicit local decisions, team policy only when declared, and evidence export for high-risk closure or PR preparation.

## Transparent contracts

The adapter report now distinguishes supported, configured-but-unverified, and unsupported outcomes. Only the declared Codex, Cursor, and generic AGENTS-compatible adapters have installer contracts. Diff-impact output now shows its heuristic confidence and path-based limitations in both JSON and human-readable modes.

## Platform scope

`get-diff-impact.sh` and `get-task-budget-advice.sh` are the supported Unix optional-tool counterparts. Other optional 2.13 tools are explicitly Windows-first until equivalent Unix contracts and tests exist.
