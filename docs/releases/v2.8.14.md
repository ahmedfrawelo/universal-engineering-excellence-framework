# UEEF 2.8.14

## Bootstrap Runtime Path Normalization

Environment bootstrap now accepts `UEEF_GLOBAL_PATH` in both supported forms: the runtime directory containing `UEEF-LOADER.md`, or its parent global runtime root. Windows and Unix bootstrap scripts normalize the path before checking mandatory runtime files.

## Regression Coverage

The bootstrap test verifies both forms with a valid temporary runtime and rejects any false missing-runtime result.
