# UEEF 2.8.6

Released: 2026-07-13

## Browser Control Routing Fix

- Chrome tasks now require the installed Chrome control skill and its `mcp__node_repl__js` browser-client path.
- The extension binding must enumerate `user.openTabs()` and claim the exact returned tab object before interaction.
- Direct Playwright MCP, Chrome DevTools MCP, in-app browser, and isolated contexts cannot substitute for the user's Chrome window.
- Window state remains untouched unless the user explicitly requests a change.
- PowerShell and Unix contract tests reject contradictory browser-routing policy.
