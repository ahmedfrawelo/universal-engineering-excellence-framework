# UEEF v2.6.0

UEEF now routes every task through an explicit complexity and risk classifier before substantial execution.

## Added

- T0 through T4 task classification with security, production, migration, architecture, and destructive-operation floors.
- Capability-based mappings for fast, balanced, and frontier Codex agents.
- Single-agent, sidecar, parallel-specialist, and independent-verifier topologies.
- A delegation-benefit test that prevents child agents from increasing tokens or latency on trivial work.
- Bounded context packets, fan-out limits, result reuse, prompt deduplication, and agent lifecycle controls.
- Automatic escalation after ambiguity, failures, scope growth, or newly discovered risk.
- `scripts/select-agent-route.ps1` for deterministic, machine-readable route decisions.

## Compatibility

Model names are defaults for the currently exposed Codex runtime. Capability classes remain authoritative when another runtime exposes different models or no override API.
