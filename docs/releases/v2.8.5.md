# UEEF 2.8.5

Released: 2026-07-12

## Summary

This patch prevents an active goal from ending a turn with an incomplete/no-result final answer instead of continuing execution.

## Changes

- Active goals use commentary for progress and continue execution.
- A goal-task final answer is allowed only for complete work, a valid external blocker, or an explicit status-only request.
- Current goal status must be read before finalization.
- `FINAL_ALLOWED` formalizes the completion predicate.
- Source, generated loader, generated `AGENTS.md`, runtime hardening, and continuation tests enforce the contract.
