# UEEF 2.11.0

Release date: 2026-07-22

## Highlights

- Fresh Codex installation and runtime synchronization now install the pinned Open Design `design-brief` and `frontend-design` skills when missing.
- The generated global instruction block is compact and points to canonical UEEF modules, reducing repeated per-turn context while preserving required activation and safety rules.
- Direct Playwright and Chrome DevTools restrictions now apply only when a task depends on the user's existing Chrome session; authorized isolated/local testing remains available.
- Continuous assurance now produces a concise default summary and exposes full diagnostics through `-Detailed`.

## Validation

- Installer and runtime hardening tests cover fresh Codex installation, idempotent skill presence, transactional runtime synchronization, and state rollback.
- Cross-platform validators, release consistency tests, and continuous assurance are required before release publication.
