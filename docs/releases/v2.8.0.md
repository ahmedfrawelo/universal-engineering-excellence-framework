# UEEF 2.8.0

Released: 2026-07-12

## Summary

This release turns agent routing from guidance into a runtime-verified contract and caps all UEEF-requested reasoning at medium.

## Changes

- Route schema version 2 emits `reasoningCeiling: medium` on Windows and Unix.
- T0 through T4 routes never emit reasoning above medium.
- T2 through T4 require bounded agent use when an independent stream has positive value, or a fixed single-agent reason.
- Active runtime state records the routing contract version and reasoning ceiling.
- Status fails when the installed routing contract is missing, stale, or exceeds the ceiling.
- Generated global loader and Codex `AGENTS.md` carry the same enforceable policy for every task.

## Verification

- PowerShell and Unix route parity tests.
- Runtime state and status contract tests.
- Full framework validation, audit, drift, installer, update, cleanup, and documentation checks.
