# UEEF v2.7.0

UEEF now integrates the five design-engineering skills published at `emilkowalski/skills`.

## Skills

- `emil-design-eng`: primary design-engineering and motion implementation guidance.
- `review-animations`: strict review of animation changes.
- `improve-animations`: read-only codebase motion audit and implementation-plan generation.
- `animation-vocabulary`: precise naming for motion effects.
- `apple-design`: fluid, gesture-driven, spring-based, Apple-style interaction guidance.

The existing `ui-ux-pro-max` and `impeccable` pair remains mandatory for general UI/UX work. The new skills are selected only when their narrower trigger applies, preventing unnecessary context and token use.

PowerShell and Unix installer scripts make the same suite reproducible on another UEEF installation.
