# UEEF 2.8.8

Released: 2026-07-13

## Cross-Session Browser Evidence Handoff

- Added a trusted coordinator evidence handoff for the exact user-owned Chrome tab when a receiving task has a degraded Node REPL or browser-client channel.
- Defined `THREAD_CONTROL_CHANNEL_DEGRADED` separately from independently proven `CHROME_EXTERNALLY_UNAVAILABLE`.
- Prohibited false `BLOCKED` transitions and Chrome restart requests based only on a task-local control failure.
- Required handoff evidence to cover the exact target tab and current code state; relevant edits require a fresh same-tab verification.
