# UEEF 2.16.1

Release date: 2026-07-23

- Reconciled the Pack 58 system, capability policy, routing gate, source loader, and generated-runtime contract: `medium` is economical default rather than a hard ceiling, and documented high-ambiguity `T3/T4` escalation remains allowed.
- A child-agent evidence record is now required only for a route that actually spawned a child. A narrow `T1` route with `NO_INDEPENDENT_WORK` is complete with its route line alone.
- Added static regression coverage for both conditions, a one-file `T1` fixture, sync recovery guidance, and Quick Start instructions for the `strict-scope` policy.
