# UEEF 2.12.1

Release date: 2026-07-23

## Highlights

- Adds `config/assurance-budgets.json` and a PowerShell assurance measurement command. It measures one audit run, checks its status and durations against explicit budgets, and can write evidence only when requested.
- Adds a cross-platform audit summary schema: `schemaVersion`, UTC generation time, normalized root, mode, status, and timed checks.
- Adds Quick/Full audit mode labeling on Unix and keeps the default human-readable output intact.
- Updates Windows CI to enforce the intentionally generous Quick budget; Unix continues to collect structured timings without an environment-sensitive enforcement gate.

## Validation

- Fixture tests prove valid, over-budget, and failed-audit summaries are classified correctly.
- Windows and Git Bash Quick audits both emit ten timed checks using schema version 1.
- Framework validators and release-consistency checks pass on Windows and Unix.
