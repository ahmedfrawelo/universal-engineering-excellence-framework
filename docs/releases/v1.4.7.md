# UEEF v1.4.7

Patch release aligning Unix Environment Bootstrap portability and automatic profile detection with PowerShell.

## Changes

- Unix uses `$HOME/.codex` when `CODEX_HOME` is not set.
- Unix detects Frontend, UIUX, Backend, Database, and DevOps signals when `UEEF_PROFILES` is not supplied.
- Explicit profile selection remains supported.
