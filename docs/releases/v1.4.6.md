# UEEF v1.4.6

Patch release adding automatic profile detection to Environment Bootstrap.

## Changes

- With no explicit profile list, bootstrap detects Frontend, Backend, Database, UIUX, and DevOps from repository signals.
- Core and AI remain mandatory for every task.
- Explicit `-Profile` or `UEEF_PROFILES` selection overrides automatic detection when task scope is not visible in files.
- The UEEF repository itself auto-detects Core and AI only, and the readiness smoke test passes.
