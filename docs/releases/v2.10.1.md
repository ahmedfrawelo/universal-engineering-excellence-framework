# UEEF 2.10.1

Release date: 2026-07-18

## Same-tab browser control-channel failover

- Added an explicit automatic failover contract for browser-control transport failures.
- Required recovery to preserve the same user-owned Chrome window, profile, signed-in session, and target tab.
- Ordered recovery as existing binding, ownership repair, trusted same-tab `VERIFIED_HANDOFF`, then verified visible Windows control only when the Chrome plugin itself is unavailable.
- Prohibited asking the user to acknowledge a handoff, type "done", reshare a tab, restart Chrome, or accept an alternate browser for task-local channel failures.
- Added contract coverage to prevent regression to manual handoff prompts or connector-created browser fallbacks.
