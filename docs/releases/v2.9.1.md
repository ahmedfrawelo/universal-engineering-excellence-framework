# UEEF 2.9.1

Release date: 2026-07-16

## Component-family ownership

- Added a strict search, reuse, extend, then create decision for shared components.
- Required one owned family folder, one canonical primitive, and one public entrypoint for related reusable files.
- Required recipes such as shell, table, list, card, and dashboard skeletons to consume the canonical primitive while owning only their structure.
- Prohibited parallel shared folders and competing imports for semantically overlapping capabilities.
- Added runtime, checklist, gate, and cross-platform validator coverage for the organization contract.
