# UEEF v1.4.4

Patch release aligning Unix Environment Bootstrap with the PowerShell implementation.

## Changes

- Unix bootstrap accepts `UEEF_PROFILES` and reports selected profiles.
- Core and AI mandatory checks are consistent across shells.
- UIUX checks both `ui-ux-pro-max` and `impeccable` when UIUX is selected.
- Unix validation requires every pack 50 module and current release notes.

## Verification

Use `UEEF_PROFILES=Core,AI,UIUX ./scripts/environment-bootstrap.sh` on Unix. On Windows, use `scripts/environment-bootstrap.ps1 -Profile Core,AI,UIUX`.
