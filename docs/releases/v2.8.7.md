# UEEF 2.8.7

Released: 2026-07-13

## Chrome Bridge Recovery and Visual Completion

- Treat transient Node REPL, browser-client, and extension bridge failures as recoverable control-channel failures.
- Require the installed bootstrap and Chrome troubleshooting guidance before giving up or considering fallback.
- Keep the same extension binding and reacquire stale tabs through `user.openTabs()` plus exact-object `claimTab()`.
- Prohibit `COMPLETE` when requested same-tab visual verification has not run.
- Build, tests, component reuse, source inspection, and structural equivalence cannot replace required browser evidence.
